# Keystroke Analyzer (Educational Project)

## 📌 Description
This project is a Python-based keystroke logging and analysis tool built for educational and cybersecurity awareness purposes.

## 🚀 Features
- Logs keystrokes with timestamps
- Counts key frequency
- Displays statistics
- Visualizes data using graphs
- Simple GUI interface

## 🛠️ Technologies
- Python
- pynput
- tkinter
- matplotlib

## ▶️ How to Run
1. Install dependencies:
   pip install pynput matplotlib

2. Run the project:
   python main.py

This project is strictly for educational use. Do not use it on systems without permission.
